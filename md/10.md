Version : 1.0

# Purpose

Dokumen ini mendefinisikan implementasi class utama Theme Plugin.

Class ini merupakan entry point seluruh theme.

Semua proses harus dimulai dari class ini.

---

# Base Class

Theme wajib mewarisi:

ThemePlugin

Tidak diperbolehkan menggunakan class lain sebagai parent.

# Plugin Name

SigerpikModernThemePlugin

Plugin Directory

# Responsibilities

Plugin hanya bertugas:

- mendaftarkan theme
- memuat asset
- memuat locale
- mendaftarkan hook
- mendaftarkan opsi theme
- bootstrap service

Plugin tidak boleh:

- melakukan query database
- menghasilkan HTML
- melakukan render Smarty
- menyimpan business logic
