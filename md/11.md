# Template Engine Specification

Version : 1.0

Project : SigerPIK Modern Theme

Platform : Open Journal Systems 3.4.x

Status : Approved

---

# Purpose

Template Engine bertugas merender seluruh tampilan Theme.

Template tidak boleh berisi business logic.

Template hanya menerima data yang sudah diproses oleh Service Layer.

---

# Design Principle

Business Logic

↓

Service

↓

Template Variables

↓

Smarty

↓

HTML

---

# Rules

Template tidak boleh:

- Query Database
- Membuat SQL
- Memanggil DAO
- Mengubah Setting
- Menulis File

Template hanya:

- Render
- Loop
- Condition sederhana
- Include Component
