# Project Bootstrap Specification

Version : 1.0

Project : SigerPIK Modern Theme

Platform : Open Journal Systems (OJS) 3.4.x

Author : SigerPIK Development Team

Status : Design Approved

---

# Purpose

Dokumen ini mendefinisikan proses bootstrap Theme SigerPIK Modern sejak plugin dimuat oleh OJS hingga halaman selesai dirender.

Bootstrap harus:

- ringan
- modular
- mudah diuji
- tidak mengubah core OJS
- kompatibel dengan update OJS

# Plugin Entry Point

SigerpikModernThemePlugin.php

Responsibilities

- bootstrap theme
- register hook
- register assets
- initialize services
- expose theme options

Must NOT

- query database directly
- contain HTML
- contain CSS
- contain business logic

# Conclusion

The Bootstrap layer is the heart of the SigerPIK Modern Theme.

All features start here.

Any future module must integrate through the Bootstrap lifecycle and never bypass the Theme Kernel.
