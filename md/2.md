Version : 1.0

Project : SigerPIK Modern Theme

Target Platform : Open Journal Systems 3.4.x

# Purpose

Dokumen ini menjelaskan arsitektur internal Theme SigerPIK Modern.

Theme harus:

- Tidak mengubah core OJS
- Upgrade Safe
- Modular
- Plugin Based
- High Performance
- Maintainable

Semua fitur wajib dibangun melalui Theme Plugin API milik PKP.
