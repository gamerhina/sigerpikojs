# SigerPIK Theme by Bihikmi

Professional Responsive Theme for Open Journal Systems (OJS) 3.4+ built for Universitas Lampung.

## Key Features

- **Modern UI**: Designed to look like top international journals (MDPI, Frontiers, Springer, Elsevier).
- **Responsive Layout**: Works flawlessly on Mobile, Tablet, and Desktop.
- **Hero Slider**: Customizable slideshow banner on the homepage.
- **Wavy Footer**: Beautiful, modern footer design with a wave border and floating actions (WhatsApp, Back-to-top).
- **Integrated Sidebar**: Custom widgets for indexing logos, visitor graphs, statistics, and social links.

## Installation

1. Copy this folder into `plugins/themes/sigerpikModern`.
2. Go to the OJS admin panel under **Settings > Website > Appearance > Theme**.
3. Choose **SigerPIK Modern Theme** and save.
